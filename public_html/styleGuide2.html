Textual Elements

There are two typefaces used: Source Sans Pro (for headings) and Inconsolata (for body text).

CSS selector:

h1, h2, p , a, li

Sample code:

<h1> Main Heading </h1>
<h2> Subpage Title </h2>
<h3> Subsection Heading </h3>
<p> This is what a paragraph would look like. <p>
<a> In-text link </a>
<ul> <li> List item <li> </ul>

Interactive Button

This type of button is used during the checkout process.

CSS selector:

a.btn, a.btn:hover, a.btn:focus, a.btn:active a.btn.bag-it, a.btn.confirm-order

Sample code:

<a href="page.html" class="btn"> Default Button </a>

Rendered element:

Default Button

Navigation Button

There are two types of navigation buttons: static and current.

CSS selector:

.btn-nav, .btn-nav:hover, .btn-nav.current

Sample code:

<a href="page.html" class="col1of6 btn-nav"> Static </a>
<a href="page.html" class="col1of6 btn-nav current"> Current </a>

Rendered element:

  • Current
  • Static

Text Fields

Textfields are used when user input is needed. These include user reviews, user comment, contact, and the checkout process.

CSS selector:

input

Sample code:

<div class="col3of6 form-input"> 
    <label for="label-name"> Label Name </label>
    <input type="text" id="first-name" placeholder="Jane">
</div>

Rendered element:

Dropdown Menus

CSS selector:

Sample code:

<label for= "province1"> </label>
<select name="province1"> 
   <option value="AL"> Alberta </option>
   ...
   <option value="SASK">Saskatchewan</option>
</select>

Navigation Bar

The navigation bar is comprised of six page links.

CSS selector:

.main-nav

Sample code:

<header>
    <a href=""><img src="img/logo.png" id="logo" alt="company logo"></a>
    <nav id="main-nav">
        <ul>
            <li><a class="col1of6 btn-nav current href=""> Home </a></li>
            ...
            <li><a class="col1of6 btn-nav href=""> Account </a></li>
        </ul>
    </nav>
</header>

  • Home
